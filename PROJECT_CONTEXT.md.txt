# 5 PECADOS MUSICALES — PROJECT CONTEXT

## VISIÓN GENERAL

5 Pecados Musicales es un proyecto artístico, performativo y transmedia compuesto por cinco avatares simbólicos: Leo Feelow, Ander Popu, Kim Thek, Bodhi Folk y Asher Coda.

No son personajes ficticios tradicionales ni alter egos psicológicos del creador. Son entidades conceptuales que representan tensiones culturales, existenciales y artísticas del mundo contemporáneo.

El proyecto mezcla:
- música
- performance
- estética cinematográfica
- dirección artística
- teoría crítica
- contenido digital
- identidad visual
- narrativa transmedia

La música es el eje base, pero el proyecto NO debe sentirse como un grupo musical convencional. Debe sentirse como un universo conceptual vivo.

El objetivo de la web NO es informar únicamente. Debe sentirse como una experiencia inmersiva, artística y cinematográfica.

---

# REGLAS GENERALES DEL UNIVERSO

- Cada avatar tiene identidad visual propia.
- Cada avatar tiene color propio.
- Las identidades NO deben mezclarse visualmente.
- El proyecto debe mantener coherencia estética absoluta.
- Evitar estética genérica IA.
- Todo debe sentirse cinematográfico, editorial y artístico.
- La experiencia debe priorizar diseño y atmósfera antes que funcionalidad.
- La web debe sentirse como una instalación digital o universo inmersivo.
- Mobile first SIEMPRE.
- La sensación debe ser oscura, profunda, elegante y conceptual.
- El minimalismo debe convivir con profundidad visual.
- El humo y la iluminación son elementos MUY importantes.
- La iluminación debe reaccionar coherentemente al color conceptual de cada personaje.
- Los efectos visuales deben sentirse reales, no artificiales.
- Evitar saturación excesiva de interfaz.
- La experiencia debe transmitir misterio, tensión y personalidad.
