# 5PM Web Structure

This folder contains the main 5PM website.

- `index.html` — landing page with hero, manifesto, avatars, and contact sections.
- `scripts/` — JavaScript modules and interaction code.
- `styles/` — compiled CSS styling for the site.
- `assets/images/` — organized image assets for the website.
- `assets/archives/` — placeholder for media or archive assets.

Notes:
- `index.html` loads `styles/styles.css` and `scripts/three-scene.js` + `scripts/app.js`.
- The `entry-gate` opener is kept as a full-screen interaction layer.
